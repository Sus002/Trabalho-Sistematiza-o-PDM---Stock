README



Stock



Problema:



O desperdício de alimentos nas residências é um grande problema, causado pela falta de controle sobre as datas de validade dos itens guardados nos mercados. Isso causa perdas financeiras e é prejudicial para o meio ambiente



Justificativa:



Este projeto está diretamente alinhado ao Consumo e Produção Responsáveis, especificamente à Meta 12.3 que é reduzir pela metade o desperdício de alimentos até 2030. Fornecendo uma ferramenta que ajuda o usuário a consumir itens antes que estraguem.



Público-Alvo:



Famílias, estudantes e quaisquer indivíduos que realizam compras semanais ou quinzenais e querem economia, organização e a redução do desperdício de alimentos em casa.



Objetivos do App:





Reduzir o desperdício doméstico de alimentos.



Permitir que o usuário cadastre itens rapidamente.



Garantir a persistência dos dados usando o Firebase.



Aplicação:



    Tipo: PWA



    Framework: Não Aplicável - HTML, CSS, JavaScript 



    Linguagem: JavaScript



Instalação e Uso:






    1. Clone o repositório do GitHub: (https://github.com/Sus002/Trabalho-Sistematiza-o-PDM---Stock)


&nbsp;   2. Abra o arquivo \*\*index.html\*\* em qualquer navegador web moderno.



&nbsp;   3. (Se necessário) Instale o PWA no seu dispositivo através do menu do navegador.



Requisitos do Sistema:



    Sistema Operacional: Android (mínimo v6.0) ou iOS (mínimo v12.0).



    Dependências: Navegador Web Moderno (Chrome, Firefox, Safari) e conexão com a internet .
